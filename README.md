# Slate 📱

Your phone, treated like a proper computer.

## Structure

- `tasker/` - Tasker profiles, tasks, and projects
- `termux/` - Termux scripts and configurations
- `configs/` - Configuration files and settings
- `scripts/` - Standalone utility scripts
- `docs/` - Documentation and setup guides

## Philosophy

Everything is version controlled. No random scripts floating around.
Every change is tracked, every automation is documented.
